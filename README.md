# Coffee Site

A simple static coffee shop website built with Next.js and TypeScript.

## Features

- Responsive landing page
- Menu page with featured beverages
- FAQ section
- Reusable layout and section components

## Project Structure

- `frontend/app` — application pages and route entry points
- `frontend/components` — reusable UI components
- `frontend/data` — menu and FAQ content
- `frontend/public/images` — static image assets
- `specs/001-coffee-site` — product requirements and planning documents

## Getting Started

1. Open the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:3000 in your browser.

## Available Scripts

Inside the `frontend` folder, you can run:

- `npm run dev` — start the development server
- `npm run build` — build the app for production
- `npm run start` — run the production build
- `npm run lint` — run ESLint

## Notes

This project is designed as a lightweight coffee shop website and can be extended with additional pages, animations, or a booking flow.
